W3C & Markup Language

To build websites, we follow certain rules and standards. Let's learn who creates those standards and what a markup language means.

What is W3C?

W3C (World Wide Web Consortium) is the organization that creates and maintains standards for the web. This ensures that websites work properly in all browsers and devices.

The World Wide Web is a technology and an information system, built upon technologies like HTTP and HTML, that is governed and developed by standards organizations, most notably the World Wide Web Consortium (W3C). It is a collection of publicly accessible documents and media linked by hypertext, which are accessed through a web browser.

✅ W3C makes web rules ✅ Ensures browsers follow same standards ✅ Makes web safe, fast, and accessible

Examples of standards created by W3C:

  • HTML
  • CSS
  • Web Accessibility Guidelines

What is a Markup Language?

A Markup Language uses tags to structure and describe content on a page.

Example: HTML is a markup language.

Markup: means “marking” content with tags to give meaning and structure.

Simple HTML (Markup) Example:

<h1>Welcome</h1>
<p>This is a paragraph written in HTML.</p>

Here, <h1> and <p> are markup tags that tell the browser how to display text.

Key Points

  • W3C makes web standards
  • HTML is a markup language
  • Markup languages use tags
