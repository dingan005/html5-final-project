HTML vs HTML5

HTML stands for HyperText Markup Language. It is used to structure web pages using elements and tags.

HTML5 is the latest version of HTML with new features for modern websites—better media support, new tags, form controls, and improved performance.

Difference Between HTML and HTML5

HTML HTML5
Older version of HTML Latest and modern HTML standard
Limited multimedia support Built-in support for <audio> and <video>
No semantic tags like header, footer Semantic tags: <header>, <footer>, <nav>, <section>, etc.
No offline storage LocalStorage & SessionStorage available
Older doctype syntax Simpler doctype: <!DOCTYPE html>

HTML Example

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN">
<html>
<head><title>Old HTML</title></head>
<body>
  <h1>Welcome</h1>
</body>
</html>

HTML5 Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>HTML5 Page</title>
</head>
<body>

<header><h1>Welcome to HTML5!</h1></header>
<main>Learning modern HTML5.</main>
<footer>Footer section</footer>

</body>
</html>

Key Features of HTML5

✅ Audio & Video support ✅ Canvas (2D graphics) ✅ Semantic tags ✅ Better form inputs (email, date, range etc.) ✅ Storage API (LocalStorage, SessionStorage)
